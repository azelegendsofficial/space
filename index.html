Penjelasan Tentang Servers

Server adalah sebuah komputer yang dibuat untuk memproses permintaan dan mengirimkan data ke komputer lain melalui jaringan. Komputer lain yang di maksud ini seperti Klien yang berupa HP atau Komputer Lain.

Fungsi Server

Fungsi Server yakni menyimpan file atau sebagai tempat penyimpanan file, seperti sftp/smb, dia itu mengizinkan orang mengambil atau menaruh sebuah file ke dalam server atau membagikan nya kepada orang lain dalam jaringan. melayani layanan atau klien, seperti layanan web, jadi servers akan memuat semua file web server termasuk file penting seperti index.html/php/ejs, script.js, stye.css, dan sebagainya kepada User atau klien yang minta request ke dalam website via ip statis atau pun domain. Layanan database, seperti kita mencari suatu produk di halaman website, contoh website nya shopee, nah saat kita cari suatu produk contoh Akun Droplet Digital Ocean maka server akan memanggil database dan mengirimkan request kepada database, lalu database akan mencarikan nama produk serta harga nya dan penjual nya, ketika dapat produk yang cocok maka database akan mengirimkan request ke server, lalu sih server akan menampilkan produk tersebut.

Sistem Operasi Apa Yang Di Pakai Server?

Server biasa nya menggunakan sistem operasi yang sangat terkenal nama nya Linux, Linux adalah sebuah sistem operasi yang open source, jadi bebas untuk di modifikasi, sistem operasi nya gratis tidak berbayar, ringan tidak berat, dan stabil di bandingkan sistem operasi windows server 2019, sistem keamanan nya lumayan ketat atau bisa di modifikasi agar lebih ketat lagi, dan tidak membutuhkan sumber daya besar seperti pada os lainnya. os linux ada macam macam dan biasa nya sering di pakai oleh server pada perusahaan besar, pertama ada Ubuntu Servers, kedua ada Red Hat Enterprise Linux, ketiga ada CentOS/Rocky Linux/AlmaLinux, keempat atau terakhir ada Debian.
